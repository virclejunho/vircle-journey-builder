CUSTOMER-LED CAMPAIGN JOURNEY BUILDER

고객이 직접 그리는
마케팅 캠페인 여정도

미팅로그를 보고 에이전트가 기획을 대신 짜는 대신, 고객사가 몇 번의 선택으로 자기 브랜드의 CRM 여정도를 직접 만들고, 그 결과물을 그대로 버클 그로스팀과 공유하는 웹 도구입니다.

레퍼런스: plus-ex Contact (선택형 위저드) + 버클 컨설팅 패키지 (여정도 비주얼)
WHY · 왜 만드나
기획 과정을 고객에게 넘기면,
더 빠르고 더 정확해집니다

고객마다 캠페인이 너무 달라서 에이전트가 미팅로그만으로 여정을 정확히 짜기는 어렵습니다. 대신 고객이 아는 것(브랜드·목표·혜택·일정)을 직접 고르게 하고, 버클은 검증된 실행 패턴(템플릿)과 콘솔 세팅 가이드를 제공합니다.

지금 — 수작업

  • 노션·피그마로 매번 처음부터 제작
  • 여정도 비주얼은 디자이너 수작업
  • 고객 의도 → 기획 사이 왕복 多
  • 재기획 때 또 처음부터
→

여정도 빌더

  • 고객이 위저드로 5분 만에 초안 생성
  • 템플릿이 여정도 비주얼을 자동 구성
  • 완성 링크가 곧 협업·소통의 시작점
  • 템플릿만 바꿔 다음 캠페인 재사용
HOW · 어떻게 동작하나
브랜드 입력 → 선택 위저드 → 여정도
1

브랜드 입력 · 시작

브랜드명만 입력하면 바로 시작합니다. 로그인·가입 없이 누구나 열어 만들 수 있어요.

2

6단계 선택 위저드

캠페인 유형 · 목표/혜택 · 기간 · 타깃/채널 · 제품 사진. plus-ex처럼 한 화면에 질문 하나씩, 고른 것에 따라 다음 질문이 달라집니다.

3

여정도 자동 생성

선택을 조합해 화면 플로우 · 메시지 · 발송 일정 · 할 일이 한 번에 그려집니다. 카드를 눌러 바로 다듬을 수 있어요.

OUTPUT · 결과물 3가지 뷰
같은 여정을, 보는 사람에 맞게
→→

📱 화면 플로우

고객이 실제로 보게 될 화면(랜딩·럭키드로우·쿠폰)이 이어지는 뷰. 컨설팅 패키지 피그마와 같은 문법 — 발표·공유에 강합니다.

📋 단계 카드

CRM 발송 회차별 카드(채널·타깃·메시지). 말풍선을 클릭해 문안을 바로 수정 — 기획·편집에 강합니다.

☐ 쿠폰 세팅☐ QA발송

🗓️ 타임라인 & 할 일

날짜 축 위에 발송·준비 시점이 놓이고, 각 지점마다 담당(버클/브랜드/함께)·기한이 붙은 투두 — 실행 관리에 강합니다.

TEMPLATES · 캠페인 템플릿
검증된 실행 패턴을 템플릿으로

버클이 실제로 운영한 캠페인 패턴(분크 럭키드로우·패밀리세일 등)을 템플릿화했습니다. 고객은 고르기만, 그로스팀은 코드 없이 템플릿 파일만 손봐 확장합니다.

🎰

게임화 이벤트데모 완성

럭키드로우·퀴즈·룰렛. QR→가입→드로우→쿠폰 + 꽝 재도전 분기까지.

🏷️

세일 / 패밀리세일구조 동일

티징→오픈→타임딜→마감. 기본·선착순 쿠폰, 타임딜 구성.

✨

신상품 런칭 / 드랍구조 동일

사전 알림→선공개/우선구매→구매→후속 리뷰.

💌

리텐션 / 재구매구조 동일

구매 후 D+3/D+7/D+14 자동 시퀀스·VIP·휴면 복구.

여정도 하나에 담기는 것들
🖥️

버클 콘솔 설정 가이드

각 단계에 "이 기능은 콘솔에서 이렇게 설정" 안내가 펼쳐집니다. 고객이 막히지 않게.

💬

메시지 톤 자동 + 직접 수정

템플릿 문안에 브랜드·혜택을 치환해 채우고, 말풍선을 눌러 바로 다듬습니다. (기존 발송 톤 학습 연동 예정)

🖼️

제품 사진 반영

업로드한 사진이 랜딩·쿠폰·메시지 화면에 자동으로 들어가 실제 느낌으로.

⬇

PDF · SVG 다운로드

PDF로 보고/공유, SVG로 피그마·일러스트레이터에서 바로 편집. 노션 의존 없음.

🔄

템플릿 전환 · 재편집

완성 후 다른 캠페인 템플릿으로 바꿔도 브랜드·기간·사진·톤은 유지. 다음 달 재기획이 빨라집니다.

🖨️

그대로 보고용

PDF·SVG로 내려받아 그대로 공유. 별도 계정·링크 없이 파일 하나로 소통합니다.
